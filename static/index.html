God of Love

聽過泰國人的月老-泰國愛神嗎？

泰國愛神又稱「三相神」，會為信眾帶來愛情、魅力、財富和地位。2020年6月 泰國愛神從曼谷來台灣，正式安座在士林夜市，並且開放給無數等待愛情與幸福的人們祈願！

為了宣傳泰國愛神安座於士林夜市，我們特別舉辦『尋找愛神代言人』活動，開放所有想要成為網紅、youtuber、明星的妳，有機會一圓成名夢想！

獲得最終優勝者，可以直接獲得10萬元獎金外，還能受邀擔任年度士林夜市觀光大使，一年擁有上千萬人次的曝光，並與知名藝人搭檔拍攝10集以上的網路節目！

『尋找愛神代言人』-2020年讓妳爆紅的最好機會！

Schedule

活動流程

參加海選
(6/1 ~ 6/30)

入圍者即可獲得限量『愛神象神』佛牌，並提供獎金1680元邀請至士林夜市愛神像前拍攝2分鐘自介影片。

網路投票
(7/10 ~ 7/31)

登入臉書帳號，一天最多可投5票，前八名人氣最高者獲得晉級機會。

八強資格淘汰賽
(8/14)

8名人氣王參與網路節目拍攝，並從節目中晉級4位進入最終決賽。

四強總決賽
(8/25)

最終決賽網路節目拍攝，挑選出前3名觀光大使

Prizes

活動獎項

第一名

0元 女神獎盃 士林夜市觀光大使一年 拍攝代言人形象照 與知名藝人合作拍攝10支以上網路節目
（給予拍攝費）

第二名

0元 女神獎杯 與知名藝人合作拍攝網路節目
（給予拍攝費）

第三名

0元 女神獎杯 與知名藝人合作拍攝網路節目
（給予拍攝費）
Description

活動辦法

[ 活動主旨 ]

藉由泰國愛神安座士林夜市推廣地區觀光，並希望所有夢想成為網紅的人能透過本活動被更多人看見！

[報名資格]

30歲以下女性，對成為YouTuber、對拍攝網路節目有興趣者，皆可報名！

[ 活動時程 ]

報名參加海選：6月1日～6月30日
通知入選者88名：6月1日～6月30日（獲得愛神＋象神佛牌＋1680元通告費，於士林夜市愛神像前拍攝照片與2分鐘影片用於網路投票）
網路投票選出8強：7月10日12:00～7月31日
八強資格淘汰賽：8月14日(拍攝網路節目 8選出4強)
四強總決賽：8月25日(拍攝網路節目 4選出3強)

[ 活動獎項 ]

第一名獲得新台幣10萬元，女神獎盃及彩帶，擔任士林夜市觀光大使一年，拍攝形象照，並設立代言人人形立牌與製作代言人圖片文宣，並與知名藝人拍攝多支網路節目，並不定時拍攝商案。（將另外給予費用）
第二名獲得新台幣5萬元，女神獎盃及彩帶，與知名藝人拍攝網路節目。（將另外給予費用）
第三名獲得新台幣3萬元，女神獎盃及彩帶，與知名藝人拍攝網路節目。（將另外給予費用）

[ 如何參加 ]

點選：我要報名
按照流程填寫個人資料
上傳一張個人全身、半身照片
完成報名後，工作人員將會通知

[ 注意事項 ]

主辦單位保留擴大、修改、取消、終止此活動以及更改活動日期的權力。
